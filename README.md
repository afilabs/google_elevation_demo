# google_elevation_demo

![Google Elevation API](https://blog.afi.io/content/images/size/w1600/2026/02/location.png " Google Elevation API")

Taken from: 
- [Who on earth uses the Google Elevation API?](https://blog.afi.io/blog/who-on-earth-uses-the-google-elevation-api/)
  
How to run the app:

1. cd into your project folder
2. Open terminal and run `npm install` (this will install the required dependencies)
3. Update the `.env` file with the URI for the companion backend app (google_elevation_backend)
4. Run `npm run dev`

This project is designed to run with [google_elevation_backend](https://github.com/afilabs/google_elevation_backend).
